# TimeClock 项目

## 环境设置

1. 确保已安装 Python 3.x
2. 创建虚拟环境：
   ```
   python -m venv venv
   ```
3. 激活虚拟环境：
   - Windows PowerShell:
     ```
     .\venv\Scripts\Activate.ps1
     ```
   - Windows CMD:
     ```
     .\venv\Scripts\activate.bat
     ```
   - Linux/Mac:
     ```
     source venv/bin/activate
     ```
4. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

## 项目结构

项目主要文件和目录：
- `venv/`: Python虚拟环境
- `requirements.txt`: 项目依赖文件
